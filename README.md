# Mute-on-unplug

A shell script to mute volume when headphones are unplugged. Volume is restored when headphones are plugged back in. 

Run the script to keep your audio to yourself. Useful for when in public places.

### Prerequisites

- acpid 2.0.26
- amixer 1.1.0
